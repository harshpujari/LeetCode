# LeetCode Solutions

Python solutions with React step-through visualizers for selected problems. 

## Structure

```
LeetCode/
├── Array/              — Array & sliding window
├── Bash/               — Bash commands 
├── BinarySearch/       — Binary search
├── BitManipulation/    — Bit manipulation
├── DynamicProgramming/ — Dynamic Programming
├── LinkedList/         — Linked lists
├── Math/               — Math & bit manipulation
├── Sorting/            — Sorting reference notes
├── String/             — Strings
├── SystemDesign/       — System design components
└── Tree/               — Binary trees
```

## Visualizers

Selected problems include a self-contained React JSX/TSX step-through visualizer — dark terminal aesthetic, no external UI libraries. Drop any `.jsx` file into [StackBlitz](https://stackblitz.com) or [CodeSandbox](https://codesandbox.io) to run it.

See [UI.md](UI.md) for the design spec.
