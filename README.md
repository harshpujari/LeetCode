# LeetCode Solutions

A curated collection of LeetCode solutions in Python, with interactive React step-through visualizers (JSX/TSX) for selected problems.

## Structure

```
LeetCode/
├── Array/          — Array & sliding window problems
├── BinarySearch/   — Binary search problems
├── LinkedList/     — Linked list problems
├── Math/           — Math & bit manipulation problems
├── Sorting/        — Sorting algorithm reference notes
├── String/         — String problems
├── SystemDesign/   — System design components
└── Tree/           — Binary tree problems
```

## Visualizers

Select problems include a self-contained React JSX/TSX step-through visualizer — a dark terminal/debugger aesthetic with no external UI libraries. See [UI.md](UI.md) for the full design spec.

To run a visualizer, drop the JSX file into any React sandbox (e.g. [StackBlitz](https://stackblitz.com) or [CodeSandbox](https://codesandbox.io)).
